DrivePrefix

Returns the prefix for block devices that will be attached to the host machines. For many hosting environments this will be sd, indicating that drives will be attached like: /dev/sda, /dev/sdb, /dev/sdc...

This may be different though for some hosting environment. XenServer for example, uses the xvd prefix and attaches drives as /dev/sda, /dev/sdb, /dev/sdc...

HostPassword
The initial host password to use when creating and/or configuring cluster nodes.
HostUsername
The initial host username to use when creating and/or configuring cluster nodes.
IsProvisionNOP
Returns true if the provisioning operation actually does nothing.
MaxParallel
The maximum number of nodes that will execute provisioning steps in parallel. This defaults to 5.
RequiresAdminPrivileges
Returns true if provisoning requires that the user has administrator privileges.
ShowStatus
Specifies whether the class should print setup status to the console. This defaults to false.
WaitSeconds
Number of seconds to delay after specific operations (e.g. to allow services to stablize). This defaults to 0.0.
