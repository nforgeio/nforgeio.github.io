Remarks

You may write and read data items from the returned queue. Writes will block when the queue is full until an item has been read, freeing a slot.

 Note
Items will be serialized internally using the current IDataConverter to bytes before actually enqueuing the item. This serialized data must be less than 64KiB.

See WorkflowQueueT for more information.
