Optionally specifies a file name pattern using standard file system wildcards like [*] and [?]. This defaults to including all files.
searchOptions (Optional)
Type: System.IOSearchOption
Optionally perform a recursive search. This defaults to TopDirectoryOnly.
zipOptions (Optional)
Type: Neon.IOStaticZipOptions
Additional options that control things like whether the files are zipped within the parent directory or whether the files are assumed to contain UTF-8 text and that Windows style CRLF line endings are to be converted to Linux compatible LF endings. You can combine options by bitwise ORing them. This defaults to None.

Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type IStaticDirectory. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Remarks
 Note
The current implementation loads the files into memory so this isn't really suitable for zipping very large files.
