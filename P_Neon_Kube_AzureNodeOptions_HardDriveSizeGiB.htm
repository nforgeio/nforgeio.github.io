Remarks

Azure StandardHDD_LRS based drives may be provisioned with one of these sizes: 32 GiB, 64 GiB, 128 GiB, 256 GiB, 512 GiB, 1TiB, 2TiB, 4TiB or 8TiB.

Azure StandardSSD_LRS based drives may be provisioned with one of these sizes: 32 GiB, 64 GiB, 128 GiB, 256 GiB, 512 GiB, 1TiB, 2TiB, 4TiB, 8TiB, 16TiB or 32TiB.

Azure PremiumSSD_LRS based drives may be provisioned with sizes: 32GiB, 64GiB, 128GiB, 256GiB, 512GiB, 1TiB, 2TiB, 4TiB or 8TiB.

 Note
This size will be rounded up to the next valid drive size for the given storage type and rounded down to the maximum allowed size, if necessary.
 Note
The Azure drive sizes listed above may become out-of-date as Azure enhances their services. Review the Azure documentation for more information about what they currently support.
