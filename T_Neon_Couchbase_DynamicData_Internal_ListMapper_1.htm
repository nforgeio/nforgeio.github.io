Remarks
 Note
This class is intended for use only by classes generated by the entity-gen build tool.

The ListMapperTEntity's primary responsibility is to listen collection change events from the JArray and relay these to the parent entity.

Thread Safety
Instance members of this type are not safe for multi-threaded operations.
