Parameters

settings (Optional)
Type: Neon.DataCouchbaseSettings
Optional Couchbase settings.
image (Optional)
Type: SystemString
Optionally specifies the Couchbase container image. This defaults to ghcr.io/neonrelease/couchbase-dev:latest or ghcr.io/neonrelease-dev/couchbase-dev:latest depending on whether the assembly was built from a git release branch or not.
name (Optional)
Type: SystemString
Optionally specifies the Couchbase container name (defaults to cb-test).
env (Optional)
Type: SystemString
Optional environment variables to be passed to the Couchbase container, formatted as NAME=VALUE or just NAME.
username (Optional)
Type: SystemString
Optional Couchbase username (defaults to Administrator).
password (Optional)
Type: SystemString
Optional Couchbase password (defaults to password).
noPrimary (Optional)
Type: SystemBoolean
Optionally disable creation of thea primary bucket index.
hostInterface (Optional)
Type: SystemString
Optionally specifies the host interface where the container public ports will be published. This defaults to DefaultHostInterface but may be customized. This needs to be an IPv4 address.
limits (Optional)
Type: Neon.XunitContainerLimits
Optionally specifies the Docker container limits to use for hosting Couchbase. Note that this method will use reasonably small default limits when this is null.
