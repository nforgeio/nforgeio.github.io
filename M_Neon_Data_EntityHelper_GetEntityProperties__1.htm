Remarks

This method is handy when manually building N1QL query strings when you want to return all entity fields but you don't want to use select * because this nests each result into a property named for the bucket. I'm not entirely sure why Couchbase does this.

 Note
This method will include all public entity properties that do not have the JsonIgnoreAttribute and this also honors the property names specified by any JsonPropertyAttribute attributes.
