BinderAttachmentAttribute
Used to tag the properties of a binder document's interface definition so that the entity-gen tool will be able to generate code that implements the INotifyPropertyChanged pattern for Couchbase Lite document attachments.
BinderDocumentAttribute
Used to tag an interface indicating that the entity-gen should generate a Couchbase Lite entity document class that that implements the INotifyPropertyChanged pattern for the document attachments.
DynamicEntity
An implementation of IDynamicEntity suitable for backing most entity implementations, including those generated by the entity-gen tool.
DynamicEntityAttribute
Used to tag an interface such that the entity-gen tool will automatically generate equivalent data model classes derived from DynamicEntity.
DynamicEntityPropertyAttribute
Used to customize code generation for a property definition within an interface tagged by DynamicEntityAttribute by the entity-gen build tool.
DynamicIncludeAttribute
Used to tag a class or enum such that the entity-gen Visual Studio build tool will automatically include the type in the generated output. This is somewhat limited, as described in the remarks.

Defines the implementation of a data entity that wraps a JSON.NET JObject to provide strongly typed properties. This is used in the Couchbase Lite extensions but may be useful for other scenarios that require future proofing.
