Workflow Properties

The Workflow type exposes the following members.

Properties
  NameDescription
Client
Returns the CadenceClient managing this workflow.
Execution
Returns the execution information for the current workflow.
IsReplaying

Indicates whether the workflow code is being replayed.

 Note
WARNING: Never have workflow logic depend on this flag as doing so will break determinism. The only reasonable uses for this flag are for managing external things like logging or metric reporting.
Logger
Returns the logger to be used for logging workflow related events.
WorkflowInfo
Returns information about the running workflow.
