Parameters

image (Optional)
Type: SystemString
Optionally specifies the NATS container image. This defaults to nkubeio/nats-streaming:latest or nkubedev/nats-streaming:latest depending on whether the assembly was built from a git release branch or not.
name (Optional)
Type: SystemString
Optionally specifies the container name (defaults to nats-streaming-test).
args (Optional)
Type: SystemString

Optional NATS-STREAMING server command line arguments.

 Note
Reasonable defaults are used when args is passed as null. These connect the NATS streaming server to the NATS core server running within the container and also configure the default MEMORY persisted store.
