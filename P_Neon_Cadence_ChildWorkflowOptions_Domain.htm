Remarks

A task list must be specified when executing a workflow. For workflows started via a typed stub, this will default to the type list specified by the [WorkflowInterface(TaskList = "my-tasklist"] tagging the interface (if any).

For workflow stubs created from an interface without a specified task list or workflows created via untyped or external stubs, this will need to be explicitly set to a non-empty value.
