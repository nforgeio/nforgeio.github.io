NeonHelperExecuteCaptureAsync Method (String, Object, NullableTimeSpan, Process)

Asynchronously starts a process to run an executable file and then waits for the process to terminate while capturing any output written to the standard output and error streams.

Remarks
 Note
If timeout is exceeded and execution has not commpleted in time then a TimeoutException will be thrown and the process will be killed if it was created by this method. Process instances passed via the process parameter will not be killed in this case.
