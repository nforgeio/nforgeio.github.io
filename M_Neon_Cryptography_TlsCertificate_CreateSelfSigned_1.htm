hostname
Type: SystemString

The hostname.

 Note
You can use include a "*" to specify a wildcard certificate like: *.test.com.
bitCount (Optional)
Type: SystemInt32
The certificate key size in bits: one of 1024, 2048, or 4096 (defaults to 2048).
validDays (Optional)
Type: SystemInt32
The number of days the certificate will be valid. This defaults to 365,000 days or about 1,000 years.
wildcard (Optional)
Type: Neon.CryptographyWildcard
Optionally generate a wildcard certificate for the subdomains of hostname or the combination of the subdomains and the hostname. This defaults to None which does not generate a wildcard certificate.
issuedBy (Optional)
Type: SystemString
Optionally specifies the issuer.
issuedTo (Optional)
Type: SystemString
Optionally specifies who/what the certificate is issued for.
