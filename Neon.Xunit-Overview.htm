Neon.Xunit
Overview

Overview

Neon publishes four packages that provide handy Xunit test fixtures and related helpers:

Neon.Xunit

includes generally useful test fixtures for ASP.NET, Containers, Docker, NATS as well as a fixture that can be used to compose fixtures including custom fixtures.

Neon.Xunit.Cadence

includes a fixture for running Cadence locally as a Docker container during for unit tests.

Neon.Xunit.Couchbase

includes a fixture for running Couchbase locally as a Docker container during for unit tests.

Neon.Kube.Xunit

is currently used internally for neonKUBE related unit testing.
