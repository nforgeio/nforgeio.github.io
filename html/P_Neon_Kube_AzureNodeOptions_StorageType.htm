AzureNodeOptionsStorageType Property

Specifies the storage type to use for any mounted drives. This defaults to StandardHDD_LRS as the lowest cost option.

 Note
You should really consider upgrading production clusters to one of the SSD based storage types.

Remarks
 Note
All virtual machine sizes support StandardHDD_LRS which is why that is the default value. Consult the consult the Azure documentation to virtual machine size specified by VmSize can support PremiumSSD_LRS.

StandardHDD_LRS specifies relatively slow rotating hard drives, StandardSSD_LRS specifies standard SSD based drives, PremiumSSD_LRS specifies fast SSD based drives. Azure recommends that most production applications deploy with SSDs.
