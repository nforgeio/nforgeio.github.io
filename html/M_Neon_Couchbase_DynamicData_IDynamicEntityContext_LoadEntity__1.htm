Exceptions
ExceptionCondition
Exception The implementation should throw an exception if the entity couldn't be loaded due to some kind of error, but not if the entity simply doesn't exist.
Remarks

Advanced entity contexts such as that implemented by the Neon Couchbase Lite extension will return a non-nullisDeletedFunc value that can be tested to check if the entity still exists.
