SshProxyTMetadataDisableSudoPrompt Method

Prevents sudo from prompting for passwords.

 Note
The connected user must already be a member of the root group.
 Note
You do not need to call Connect(TimeSpan) or WaitForBoot(NullableTimeSpan, Boolean) before calling this method (in fact, calling those methods will probably fail).
