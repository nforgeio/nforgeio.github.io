Remarks

Advanced entity persistence frameworks, like the Neon Couchbase Lite extensions, implement this interface to provide allowing entities to link to one another. Links are specified as opaque strings. For Couchbase Lite, these will be document IDs. Other frameworks can specify something else.

LoadEntityTEntity(String, FuncBoolean) attempts to load an entity, given its link. The method returns null if the linked entity doesn't exist and will throw an exception if there was some other kind of error.

LoadDocumentTDocument(String, FuncBoolean) attempts to load a document, given its link. The method returns null if the linked document doesn't exist and will throw an exception if there was some other kind of error.
