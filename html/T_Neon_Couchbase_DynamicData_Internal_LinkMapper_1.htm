Remarks
 Note
This class is intended for use only by classes generated by the entity-gen build tool.

This class is used to link a JProperty value to an external entity. The property value will act as the entity link and the IDynamicEntityContext passed to the constructor (if any) will be used to dereference the link and load the entity.

Linked entities are loaded on demand and cached when the Value getter is called. Subsequent calls to the getter will return the cached value. The getter will return null if the link is null or if the referenced entity doesn't exist.

 Note
This class will simply return null if no IDynamicEntityContext is present.
Thread Safety
Instance members of this type are not safe for multi-threaded operations.
