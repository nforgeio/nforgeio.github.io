Parameters

transientDetector (Optional)
Type: SystemFuncException, Boolean
Optionally specifies the function that determines whether an exception is transient (see TransientDetector). You can pass null if all exceptions are to be considered to be transient.
maxAttempts (Optional)
Type: SystemInt32
Optionally specifies the maximum number of times an action should be retried (defaults to 5).
initialRetryInterval (Optional)
Type: SystemNullableTimeSpan
Optionally specifies the initial retry interval between retry attempts (defaults to 1 second).
maxRetryInterval (Optional)
Type: SystemNullableTimeSpan
Optionally specifies the maximum retry interval (defaults to essentially unlimited: 24 hours).
timeout (Optional)
Type: SystemNullableTimeSpan
Optionally specifies the maximum time the operation will be retried (defaults to unconstrained)
sourceModule (Optional)
