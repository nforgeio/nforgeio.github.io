SshProxyTMetadata Properties

The SshProxyTMetadata generic type exposes the following members.

Properties
  NameDescription
Cluster
The associated ClusterProxy or null.
ConnectTimeout
The connection attempt timeout. This defaults to 5 seconds.
DefaultRunOptions
Specifies the default options to be bitwise ORed with any specific options passed to a run or sudo execution command when the Defaults flag is specified. This defaults to None.
DownloadFolderPath
Returns the path to the user's download folder on the server.
FileTimeout
The file operation timeout. This defaults to 30 seconds.
HomeFolderPath
Returns the path to the user's home folder on the server.
IsFaulted
Indicates that the server is in a faulted state because one or more operations have failed.
IsReady
Indicates that the server has completed or has failed the current set of operations.
Metadata
Applications may use this to associate metadata with the instance.
Name
Returns the display name for the server.
OsName

Returns the name of the remote operating system (e.g. "Ubuntu").

 Note
This is only valid after a connection has been established.
OsVersion

Returns the version of the remote operating system (e.g. "18.04.1").

 Note
This is only valid after a connection has been established.
PrivateAddress
Returns the cluster private IP address to used for connecting to the server.
PublicAddress
Returns the cluster public IP address, FQDN, or null for the server.
RemotePath
The PATH to use on the remote server when executing commands in the session or null/empty to run commands without a path. This defaults to the standard Linux path and Bin.
RetryCount
The number of times to retry a failed remote command. This defaults to 5.
SshPort
The SSH port. This defaults to 22.
Status
The current server status.
UploadFolderPath
Returns the path to the user's upload folder on the server.
Username
Returns the username used to log into the remote node.
