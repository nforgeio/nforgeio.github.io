Parameters

stepLabel
Type: SystemString
Brief step summary.
nodeAction
Type: SystemActionSshProxyNodeMetadata, TimeSpan
The action to be performed on each node. Two parameters will be passed to this action: the node's SshProxyTMetadata and a TimeSpan indicating the amount of time the action should wait before performing the operation, if the operation hasn't already been performed.
nodePredicate (Optional)
Type: SystemFuncSshProxyNodeMetadata, Boolean
Optional predicate used to select the nodes that participate in the step or null to select all nodes.
quiet (Optional)
Type: SystemBoolean
Optionally specifies that the step is not to be reported in the progress.
noParallelLimit (Optional)
Type: SystemBoolean
Optionally ignores the global MaxParallel limit for the new step when greater.
stepStaggerSeconds (Optional)
Type: SystemInt32
Optionally specifies the time delay used to stagger execution of the nodes executing this step. Setting a non-zero value of perhaps 5 seconds will help mitigate problems with multiple accessing nodes trying to download the same files from the Internet at the same time, potentially causing the remote endpoint to start throttling access.
position (Optional)
Type: SystemInt32
Optionally specifies the zero-based index of the position where the step is to be inserted into the step list.
parallelLimit (Optional)
Type: SystemInt32
Optionally specifies the maximum number of operations to be performed in parallel for this step, overriding the controller default.
