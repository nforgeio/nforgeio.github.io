AzureNodeOptions Properties

The AzureNodeOptions type exposes the following members.

Properties
  NameDescription
HardDriveCount

Specifies the number of managed Azure data drives to attach to the node's virtual machine. This defaults to 1.

 Note
Currently only values of 0..1 are supported. In the future we may allow multiple data disks to be mounted and combined into a RAID0 array.
HardDriveSizeGiB
Specifies the size of each of the mounted managed drives in gigabytes. This defaults to 64GiB.
StorageType

Specifies the storage type to use for any mounted drives. This defaults to StandardHDD_LRS as the lowest cost option.

 Note
You should really consider upgrading production clusters to one of the SSD based storage types.
VmSize
Specifies the Azure virtual machine size. This defaults to Standard_DS3_v2.
