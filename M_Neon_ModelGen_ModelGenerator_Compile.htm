Remarks

By default, this method will compile the assembly with references to .NET Standard 2.0.

You may customize these by passing a referenceHandler action. This is passed the list of MetadataReference instances. You can add or remove references as required. The easiest way to add a reference is to use type reference like:

Copy
using Microsoft.CodeAnalysis;

...

var source   = "public class Foo {}";
var assembly = ModelGenerator.Compile(source, "my-assembly",
    references =>
    {
        references.Add(typeof(MyClass));    // Adds the assembly containing MyClass.
    });
