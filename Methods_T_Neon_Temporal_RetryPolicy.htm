DoNotRetry(String)

Appends the string passed to NonRetriableErrors as a reason not to be retried.

 Note
We generally recommend that you use DoNotRetryExceptionType by default and reserve this for situations where you need to specify a specific reason, probably for interoperating with workflows and activities written in other lanagues, etc.
DoNotRetryExceptionType

Appends the error reason corresponding to an exception type. For built-in Temporal exceptions, this will append the apporpriate reason string and for other exception type, this will append the fully qualified exception type name.

 Note
