Remarks

This folder will exist on developer/operator workstations that have used the neon-cli to deploy and manage clusters. Each known cluster will have a JSON file named NAME.context.json holding the serialized KubeContextExtension information for the cluster, where NAME maps to a cluster configuration name within the kubeconfig file.
