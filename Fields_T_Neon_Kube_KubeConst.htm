KubeConst Fields

The KubeConst type exposes the following members.

Fields
  NameDescription
ContainerGID

The default host machine container group ID.

 Note
This explictly set to the first valid normal Linux user ID to be compatible with as many Docker images as possible.
ContainerGroup
The default host machine container group name.
ContainerUID

The default host machine container user ID.

 Note
This explictly set to the first valid normal Linux user ID to be compatible with as many Docker images as possible.
ContainerUser
The default host machine container username.
DefaultPassword
The default password for component dashboards and management tools (like Ceph and RabbitMQ).
DefaultUsername
The default username for component dashboards and management tools (like Ceph and RabbitMQ).
DefaulVmTemplatePassword
The root account password baked into the Hyper-V and XenServer cluster host node virtual machine templates. Note that this will not be used for hosts provisioned on public clouds for security reasons.
DefaulVmTemplateUsername
The root account username baked into the Hyper-V and XenServer cluster host node virtual machine templates. This is also used as the username for hosts provisioned to clouds like Azure, Aws, and Google Cloud.
DesktopServicePort
The default local network port for the neonKUBE desktop API used by the neon-cli tool for communicating with the neonKUBE desktop.
DockerPublicRegistry
Hostname of the Docker public registry.
GrafanaDashboardProxyPort
The default local network port used for proxying requests to the Grafana dashboard for the current cluster.
KialiDashboardProxyPort
The default local network port used for proxying requests to the Kiali dashboard for the current cluster.
KibanaDashboardProxyPort
The default local network port used for proxying requests to the Kibana dashboard for the current cluster.
KubectlProxyPort
The default local network port where kubectl proxy will listen and forward traffic to the Kubernetes API server.
KubeDashboardProxyPort
The default local network port used for proxying requests to the Kubernetes dashboard for the current cluster.
LocalStorageClassName

The default name for the local V1StorageClass

LocalVolumePath

The default path for the LocalStorageClassName

 Note
This is temporary, once Kubernetes supports dynamic provisioning of local storage volumes, we'll use that instead.
MaxJitter
Timespan used to introduce some random jitter before an operation is performed. This is typically used when it's possible that a large number of entities will tend to perform an operation at nearly the same time (e.g. when a message signalling that an operation should be performed is broadcast to a large number of listeners. Components can pass this to PseudoRandomTimespan(TimeSpan) to obtain a random delay timespan.
MaxMasters
The maximum number of cluster master nodes.
MinMasterCores
The minimum number of cores required by master nodes.
MinMasterNics
The minimum required network interface cards for master nodes.
MinMasterRamMiB
The minimum RAM (MiB) required for master nodes.
MinWorkerCores
The minimum number of cores required by worker nodes.
MinWorkerNics
The minimum required network interface cards for worker nodes.
MinWorkerRamMiB
The minimum RAM (MiB) required for worker nodes.
NeonDevRegistry
Identifies the development cluster public Docker registry.
NeonProdRegistry
Identifies the production cluster public Docker registry.
PrometheusDashboardProxyPort
The default local network port used for proxying requests to the Prometheus dashboard for the current cluster.
RootUser
The root Kubernetes context username for provisioned clusters.
SysAdminGID
The default host machine sysadmin group ID.
SysAdminGroup
The default host machine sysadmin group.
SysAdminUID
The default host machine sysadmin user ID.
SysAdminUser
The default host machine sysadmin username.
TestModeFolderVar
The environment variable used for unit testing that indicates that KubeHelper should run in test mode. The value will be set to the path of the temporary directory where the Kubernetes and neonKUBE files will be located.
