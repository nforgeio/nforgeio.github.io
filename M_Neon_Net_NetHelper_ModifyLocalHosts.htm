NetHelperModifyLocalHosts Method

Used to temporarily modify the hosts file used by the DNS resolver for debugging or other purposes.

 Note
WARNING: Modifying the hosts file will impact all processes on the system, not just the current one and this is designed to be used by a single process at a time.

hostEntries (Optional)

A dictionary mapping the hostnames to an IP address or null.
section (Optional)
Type: SystemString

Optionally specifies the string to use to mark the hostnames section. This defaults to MODIFY which will delimit the section with # NEON-BEGIN-MODIFY and # NEON-END-MODIFY. You may pass a different string to identify a custom section.

 Note
The string passed must be a valid DNS hostname label that must begin with a letter followed by letters, digits or dashes. The maximum length is 63 characters.
Remarks
 Note
This method requires elevated administrative privileges.

This method adds or removes a temporary section of host entry definitions delimited by special comment lines. When hostEntries is non-null and non-empty, the section will be added or updated. Otherwise, the section will be removed.

You can remove all host sections by passing both hostEntries and section as null.
