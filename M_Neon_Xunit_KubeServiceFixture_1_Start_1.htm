Parameters

serviceCreator (Optional)
Type: SystemFuncTService
Callback that creates and returns the new service instance.
runningTimeout (Optional)
Type: SystemTimeSpan
Optionally specifies the maximum time the fixture should wait for the service to transition to the Running state. This defaults to 30 seconds.

Return Value

Type: TestFixtureStatus
Started if the fixture wasn't previously started and this method call started it or AlreadyRunning if the fixture was already running.
Exceptions
ExceptionCondition
TimeoutException Thrown if the service didn't transition to the running (or terminated) state within runningTimeout.
Remarks

This method first calls the serviceCreator callback and expects it to return a new service instance that has been initialized by setting its environment variables and configuration files as required. The callback should not start the service.
