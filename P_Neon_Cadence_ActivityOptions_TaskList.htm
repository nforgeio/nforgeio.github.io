Remarks

A task list must be specified when executing an activity. For activities started via a typed stub, this will default to the type list specified by the [ActivityInterface(TaskList = "my-tasklist"] tagging the interface (if any).

For activity stubs created from an interface without a specified task list or activities created via untyped or external stubs, this will need to be explicitly set to a non-empty value.
