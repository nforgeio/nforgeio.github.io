OnePassword Methods

The OnePassword type exposes the following members.

Methods
Â  NameDescription
ClearAwsCredentials

Removes the AWS-CLI credential environment variables if present:

  • AWS_ACCESS_KEY_ID
  • AWS_SECRET_ACCESS_KEY
GetAwsCredentials

Retrieves the AWS-CLI NEON_OP_AWS_ACCESS_KEY_ID and NEON_OP_AWS_SECRET_ACCESS_KEY credentials from 1Password and sets these enviroment variables:

  • AWS_ACCESS_KEY_ID
  • AWS_SECRET_ACCESS_KEY
GetPassword
Returns a named password from the current user's standard 1Password vault like [user-sally] by default or a custom named vault.
GetValue
Returns a named value from the current user's standard 1Password vault like [user-sally] by default or a custom named vault.
ReadPasswordFromConsole
Prompts the user for the master 1Password when the NC_OP_MASTER_PASSWORD environment is not set. This can be used for tools that may be run directly by the user and will need the user to manually enter the password as well as by automated scripts where the password is already present as the environment variable.
Signin
Signs into 1Password so you'll be able to access secrets.
Signout
Signs out from 1Password by removing the session environment variable.
