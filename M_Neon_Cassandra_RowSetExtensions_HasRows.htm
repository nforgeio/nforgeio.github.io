RowSetExtensionsHasRows Method

Distructively tests a RowSet to see if it has any rows. It does this by trying to fetch the first row and returning true when there was a row or false when there wasn't.

This means that if you enumerate the rows after calling this, that the first row returned by the database won't be included in the enumeration. This is what we mean by distructive.
